    About the Flags

    This tool ports the structural checks from scripts/summary_qa.py so you can triage one summary at a time without installing Python. It catches the same failure modes we saw in the Qwen-7B fine-tuning: garbled years (21020, 2k00), raw court-filing artifacts, repetition loops, meta-prompt leakage, and the Clearinghouse required-element checklist.

    Flag code legend (click to expand)
    EXTREMELY_SHORT · TOO_SHORT · TOO_LONGWord count is outside the 80–2000 word window.
    UNTERMINATED · ELLIPSIS_ENDSummary does not end with terminal punctuation — likely hit a token limit mid-sentence.
    GARBLED_DATEAlphanumeric token in a date context that looks like a botched year (2k00, 2oo4, 21020). This was the ckpt-3690 overfitting fingerprint.
    IMPLAUSIBLE_YEAR4-digit year outside the 1700 – next year range in a date context.
    RAW_DOCUMENT_ARTIFACTPleading caption, signature block, line-numbered transcript text — the model regurgitated source instead of summarizing.
    EXCESSIVE_CAPSLong run of ALL-CAPS words — usually a court header bled through.
    DUPLICATE_SENTENCE · REPETITION_LOOPThe same long sentence or 8-word phrase appears multiple times — model is stuck.
    META_PROMPT_LEAK"As an AI language model…", "Here is the summary…", "Based on the provided documents…" — meta-talk that shouldn't appear in a Clearinghouse summary.
    WRONG_JUDGMENT_SPELLINGjudgement → judgment in U.S. legal usage.
    WRONG_TENSEPresent-tense narrative verbs dominate — Clearinghouse style is past tense for procedural history.
    SUSPICIOUS_SPELLINGTokens with triple-repeated letters or unusual clusters — often overfitting typos.
    MISSING_FILING_DATENo filing-date cue found (required element).
    MISSING_COURTNo court name detected (required element).
    MISSING_STATUTENo statutory or constitutional basis detected (required element).
    MISSING_REMEDYNo remedy or relief language detected (recommended).
    MISSING_OUTCOMENo disposition or outcome language detected (recommended).
    MISSING_ACTION_TYPEDid not specify class vs. individual action (recommended).
    LENGTH_TOO_SHORT_VS_REF · LENGTH_TOO_LONG_VS_REFGenerated length is <40% or >250% of reference — only fires when a reference is provided.
    UNCITED_STATUTEReference cites a statute the summary doesn't mention. Only fires with a reference.
    UNSUPPORTED_STATUTESummary cites a statute the reference doesn't mention — possible hallucination. Only fires with a reference.
