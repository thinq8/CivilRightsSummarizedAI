## What You Need

1. **Git** and **Python 3.11+** (most Macs have both; on Windows install from python.org)
2. A terminal (Terminal on Mac, Command Prompt or PowerShell on Windows)
3. About 500 MB of free disk space
4. Optionally: a Claude/ChatGPT account (free tier works) if you want to judge summary quality


---

## Step 1: Download and Install (5 minutes)

Open your terminal and paste these commands one at a time:

```bash
# Download the project
git clone https://github.com/thinq8/CivilRightsSummarizedAI.git
cd CivilRightsSummarizedAI

# Create an isolated Python environment so nothing conflicts
python3 -m venv .venv
source .venv/bin/activate

# Install the project and its dependencies
python -m pip install --upgrade pip "setuptools<82" wheel
python -m pip install -e ".[dev]"
```

To confirm everything worked:

```bash
pytest -q
```

You should see: **4 passed**

## Step 4: Evaluate Summary Quality

This is where you can see how well our fine-tuned model summarizes civil rights
cases compared to expert-written reference summaries.

You have two options depending on what tools you have available.

### Option A: Use Claude to Judge Summaries (no GPU needed)

This is the easiest path. You just need a Claude account (claude.ai).

1. Open the file `eval_results.jsonl` (we will provide this file with our
   final submission -- it contains the model's summaries alongside the expert
   references for 50 test cases).

2. For any record, copy the **prediction** (our model's output) and the
   **reference** (the expert summary) into Claude with a prompt like:

   > I have two summaries of the same civil rights court case. Summary A was
   > written by a legal expert. Summary B was generated by an AI model.
   > Please compare them on:
   > 1. Factual accuracy -- does B contain any facts not in A?
   > 2. Completeness -- does B cover the key holdings and outcomes?
   > 3. Clarity -- is B well-organized and readable?
   > 4. Overall quality -- rate B from 1-5 (1=poor, 5=expert-level)
   >
   > **Expert summary (A):**
   > [paste reference here]
   >
   > **AI summary (B):**
   > [paste prediction here]

3. Claude will give you a detailed quality assessment. Repeat for as many
   samples as you like. Even 5-10 gives a good sense of model quality.

### Option B: Run the Evaluation Script (requires a GPU)

If you have access to a machine with a GPU (a gaming PC with an NVIDIA card
works, or Google Colab with a free GPU runtime), you can run the full
automated evaluation.

**What you need for this option:**
- A computer with an NVIDIA GPU (16+ GB VRAM) or a Google Colab notebook
- The model weights folder (we will share a download link)
- The test data file (included in the shared data)

**Steps:**

```bash
# Make sure you're in the project folder with the environment activated
cd CivilRightsSummarizedAI
source .venv/bin/activate

# Install the ML dependencies and ROUGE scoring
pip install -e ".[ml]"
pip install rouge-score

# Run evaluation on 5 samples first to make sure it works (~5 min)
python scripts/eval_checkpoint.py \
    --checkpoint-dir path/to/model-weights \
    --test-file path/to/test.jsonl \
    --num-samples 5

# If that works, run the full 50-sample evaluation (~30-60 min)
python scripts/eval_checkpoint.py \
    --checkpoint-dir path/to/model-weights \
    --test-file path/to/test.jsonl \
    --num-samples 50 \
    --output-file eval_results.jsonl
```

Replace `path/to/model-weights` and `path/to/test.jsonl` with wherever you
saved the downloaded files.

**What the script produces:**

| File | What's in it |
|------|-------------|
| Terminal output | Live progress showing each summary as it generates |
| `eval_results.jsonl` | Every test case with the model's summary and the expert reference |
| `eval_results.summary.json` | Overall scores: ROUGE-1, ROUGE-2, ROUGE-L, and average lengths |

**Reading the scores:**
- **ROUGE-1** measures word overlap (higher = more of the same words used)
- **ROUGE-2** measures phrase overlap (higher = similar phrasing)
- **ROUGE-L** measures longest matching sequences (higher = similar structure)
- Scores range from 0 to 1. For legal summarization, 0.35+ on ROUGE-1 is solid.
